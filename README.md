# aoc

My go at solving Advent of Code problems with Python

Advent of code [https://adventofcode.com/](https://adventofcode.com/)

## Requirements:
- Python 3.12

## Structure

Problem statements are named like this: `1.md`
Code solutions are named like this: `1.py` and `1.2.py`
Data files are named like this: `1.example.txt` (example from the problem statement) and `1.input.txt` (actual input data)

Where 1 is the day number
